# Missing Assets

These are the image/icon files needed to complete the website.
Please place each file in the path shown.

---

## Icons

| File path | Description |
|-----------|-------------|
| `assets/icons/favicon.ico` | The church favicon (displayed in browser tabs, header logo, footer logo). Also used as the fallback logo image throughout the site. You can grab this directly from the existing website at `https://www.saralandbaptistchurch.org/favicon.ico`. |

---

## Images

| File path | Description | Used on |
|-----------|-------------|---------|
| `assets/images/church-exterior.jpg` | Photo of the Saraland Baptist Church building (the original site shows it photographed at night — any good exterior shot works). Recommended size: **1600×900px** or larger. | Home page hero background, Home "About" section, About page |
| `assets/images/campfire-gathering.jpg` | Photo of a church community gathering around a campfire (shown on the original homepage). Recommended size: **1600×700px** or larger. | Home page community section |
| `assets/images/pastor-todd.jpg` | Headshot or portrait of Pastor Todd Richardson. Displayed in a **90×90px** circular crop on the About / Who We Are page. Any clear, friendly portrait photo works. | About page — Leadership section |

---

---

## Notes

- All images should be **JPEG or WebP** format for best performance.
- The site is designed to gracefully degrade — missing images will show the section
  without a background photo rather than breaking the layout.
- The favicon from the original Squarespace site is typically at:
  `https://www.saralandbaptistchurch.org/favicon.ico`
  — you can download it directly from there.
